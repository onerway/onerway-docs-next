## Payment UIs

Use prebuilt payment UIs to accept payments online.

| Integration | Description | Effort |
|-------------|-------------|:------:|
| **Pay by link** | Use Payment Links to share a link to accept payments without a website. | ★☆☆☆ |
| **Hosted checkout** | Use Onerway Checkout to redirect to a prebuilt payment form. | ★★☆☆ |
| **In-app payments** | Use Onerway SDK to accept payments in iOS, Android, or React Native apps. | ★★★★ |
| **API** | Use Onerway API to customize payment forms for collecting payment information. | ★★★★ |
