## 前提条件

开始使用 Onerway API 之前，需要：

- 创建用于测试的沙盒账户
- 获取 API 凭证（商户号、密钥和应用 ID）
- 配置 :prose-annotation{annotation="用于请求 Onerway API" text="Base URL"}

## 初始配置

::prose-accordion
---
type: multiple

  :::prose-accordion-item{icon="i-lucide-circle-help"}

  #label
  ### 创建沙盒账户

  #content
  创建沙盒账户时，提供 :prose-annotation{annotation="用于接收登录凭证" text="邮箱"} 和集成支付的 :prose-annotation{annotation="用于验证支付请求来源" text="域名"}。Onerway 通常在 1–2 个工作日内完成测试环境配置。

  :::

  :::prose-accordion-item{icon="i-lucide-circle-help"}

  #label
  ### 获取 API 凭证

  #content
  沙盒账户就绪后，登录 [Sandbox 商户后台](https://sandbox-portal.onerway.com/user/login){glossary="sandbox"} 获取 API 凭证：

    ::::warning
    切勿在客户端代码中暴露密钥，也不要提交到版本控制。使用环境变量或平台的密钥管理服务。
    ::::

  - **[商户号](https://sandbox-portal.onerway.com/account/info)** — 账户标识
  - **[密钥](https://sandbox-portal.onerway.com/account/info)** — API 认证密钥
  - **[应用 ID (App ID)](https://sandbox-portal.onerway.com/application/list)** — 店铺标识

  :::

  :::prose-accordion-item{icon="i-lucide-circle-help"}

  #label
  ### 配置 Base URL

  #content
  根据环境配置对应的 Base URL：

    ::::note
    开发和测试期间始终使用 Sandbox URL，避免处理真实交易。切勿混用 Sandbox 凭证和 Production URL（反之亦然）。
    ::::

  | 环境 | Base URL |
  | ----------- | -------- |
  | **Sandbox**     | `https://sandbox-acq.onerway.com`  |
  | **Production**  | `https://acq.onerway.com`          |

  环境变量配置示例：

  ```yaml [env.yaml]
  ONERWAY_BASE_URL=https://sandbox-acq.onerway.com
  ```

  :::
