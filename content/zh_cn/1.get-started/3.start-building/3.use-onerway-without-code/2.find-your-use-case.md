## 设计你的无代码集成

Onerway 为流行的电商平台提供预构建插件。在你的商店管理面板中安装并配置插件即可接受支付——无需编写代码。

::warning
在使用这些集成之前，你必须：

1. 注册 Onerway 账户
2. 提交你的商店 URL 进行验证
3. 访问后台配置你的支付设置
::
