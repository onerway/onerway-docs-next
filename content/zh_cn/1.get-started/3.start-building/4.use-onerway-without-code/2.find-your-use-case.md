## 设计你的无代码集成

Onerway 为流行的电商平台提供预构建插件。在你的商店管理面板中安装并配置插件即可接受支付——无需编写代码。

::warning
在使用这些集成之前,你必须:

1. 注册 Onerway 账户
2. 提交你的商店 URL 进行验证
3. 访问后台配置你的支付设置
::

## 电商平台集成

### 集成模式说明

所有平台支持两种集成模式:

- **卡支付(嵌入式)**: 收银台直接嵌入在你的商店结账页面中。客户可以安全地输入卡信息,无需跳转。
- **本地支付(聚合式)**: 支持卡支付和多种本地支付方式。客户会跳转到 Onerway 收银台页面,在那里可以选择他们偏好的支付选项。

### 平台对比

| 平台 | 嵌入式 | 聚合式 | 特殊要求 |
|----------|:--------:|:----------:|---------------------|
| :icon{name="platforms:shopify" class="inline size-5"} Shopify | ✓ | ✓ | 需要提供原始商店 URL |
| :icon{name="platforms:shoplazza" class="inline size-5"} Shoplazza | ✓ | ✓ | 需要提供原始商店 URL |
| :icon{name="platforms:shopline" class="inline size-5"} Shopline | ✓ | ✓ | 需要提供原始商店 URL |
| :icon{name="platforms:shopyy" class="inline size-5"} Shopyy | ✓ | ✓ | 在后台选择模式 |
| :icon{name="platforms:shoplus" class="inline size-5"} Shoplus | ✓ | — | 仅支持嵌入式 |
