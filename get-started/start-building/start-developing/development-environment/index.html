设置你的开发环境

v1

设置沙盒账户，获取 API 凭证，配置基础 URL，为您的应用集成 Onerway 做准备。

前提

在开始使用 Onerway API 构建应用之前，您需要：

  • 创建用于测试的沙盒账户
  • 获取您的 API 凭证（商户号、秘钥和应用 ID）
  • 配置请求域名

初始设置

创建沙盒账户

要创建沙盒账户，请提供您的邮箱和将要集成支付的域名。Onerway 通常会在 1-2 个工作日内为您设置好测试环境。

获取您的 API 凭证

沙盒账户准备就绪后，登录商户后台获取您的API 凭证：

永远不要在客户端代码中暴露您的秘钥，也不要将其提交到代码仓库。请使用环境变量或您平台的秘钥管理器。
  • 商户号 - 账户标识符
  • 秘钥 - API 认证密钥
  • 应用 ID (App ID) - 店铺标识符

设置请求域名

为您的环境配置适当的请求域名：

开发和测试期间始终使用沙盒 URL，以避免处理真实交易。不要将沙盒凭证与生产环境 URL 混合使用（反之亦然）。
环境基础 URL
沙盒环境https://sandbox-acq.onerway.com
生产环境https://acq.onerway.com

环境变量配置示例：

env.yaml
ONERWAY_BASE_URL=https://sandbox-acq.onerway.com
